Auto Spa CRM v15

Добавлено:
- новый раздел /booking/ — календарь записи;
- создание, редактирование и удаление записи;
- выбор клиента, автомобиля, услуги, даты и статуса;
- создание заказ-наряда из записи;
- новый пункт меню «Запись».

После установки выполните:
  docker compose exec web python manage.py migrate
если таблица crm_booking не создастся автоматически.
