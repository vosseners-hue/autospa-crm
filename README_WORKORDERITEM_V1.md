# WorkOrderItem v1

Добавляет фундамент для заказ-наряда 2.0:

- скидка на отдельную строку заказ-наряда;
- комментарий к каждой работе;
- дата создания строки;
- расчет суммы строки с учетом скидки.

После применения patch нужно выполнить миграции:

```bash
docker compose exec web python manage.py migrate
```
